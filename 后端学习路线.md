- 目标：
	- 熟练掌握 Java 常见集合
	- 掌握 Java 各种特性，如String为什么只能用`.equals()` 方法判断
	- 了解反射机制、注解

- 重点
	 - Java 基础语法
	 - 面向对象的思想
	 - Java 类的基本结构
	 - Java 泛型 (**Important**)
	 - Java 数据结构实现（**Very Important**）
	 - Java Collection类 **(Very Important**, 青空会带着看源码, 好好看看)
	 - Java Stream **(Important)**  , 能减少代码量提高可读性。
	 - 反射和注解  **(Important)**
	 - IO操作

- 作业
用 Java 做一些 Leetcode  简单题练练熟练度
